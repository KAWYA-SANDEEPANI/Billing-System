# Advanced Billing System (HTML, CSS, JavaScript)

A simple and responsive **Invoice/Billing System** built using HTML, CSS, and JavaScript.  
This project allows users to add items, calculate totals, apply discounts, and print invoices.

---

## Features

-  Add items (name, price, quantity)
-  Auto calculation of subtotal, tax, discount, and grand total
-  Delete items from bill
-  Save data using localStorage (data stays after refresh)
-  Auto-generated invoice number and date
-  Print invoice option
-  Responsive and clean UI

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API
